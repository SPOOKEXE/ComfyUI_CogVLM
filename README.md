# ComfyUI_CogVLM

CogVLM captioning

Note: this code only supports CUDA (NVIDIA) and needs xformers support/enabled.

# Installation

1. Enter you ComfyUI/custom_nodes folder
2. `git clone https://github.com/SPOOKEXE/ComfyUI_CogVLM`
3. Restart ComfyUI
